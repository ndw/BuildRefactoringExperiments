FeatureKeys
===========

The files in this directory are concerned with the management of Saxon configuration options.

The master repository of configuration parameters is in FeatureKeys.xml; this holds an XML description
of each parameter.

XSLT stylesheets are used to generate:

* Java modules defining the configuration parameters in the Java "source" code

* C# modules, similarly

* User documentation of configuration parameters

* A schema for configuration files

* A sample configuration file
